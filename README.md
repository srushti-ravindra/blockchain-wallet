# Blockchain Wallet Smart Contract

## Project Description
A simple Ethereum smart contract wallet that allows users to deposit Ether, withdraw funds, and check their balance.

## Features
- Deposit Ether into wallet
- Withdraw Ether safely
- Check user balance
- Uses Solidity mapping for storage

## Technologies Used
- Solidity
- Remix IDE
- Ethereum Smart Contracts

## Learning Outcome
Understood basics of blockchain, smart contracts, and decentralized financial systems.
